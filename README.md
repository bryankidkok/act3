## mi promp

#Genera un proyecto en Flutter organizado con rutas nombradas para las secciones vuelos, escalas y empleados. Divide el código en archivos con un main.dart para la configuración, un drawer_menu.dart para el menú lateral y una carpeta llamada paginas que contenga los archivos individuales de cada sección, usando una paleta de colores azul marino corporativo para un aeropuerto.

El drawer_menu.dart debe incluir un UserAccountsDrawerHeader con fondo azul, un avatar circular desde red y el nombre de la empresa, mostrando en el campo de texto la dirección, teléfono y email en líneas separadas. Debajo, crea los ListTile con iconos de aviación que al hacer clic cierren el menú y naveguen a la ruta correspondiente usando Navigator.pushNamed para cada una de las 4 opciones solicitadas.

Cada página de destino dentro de la carpeta paginas debe tener un AppBar azul y el contenido del body totalmente centrado. Es obligatorio incluir una imagen de 200x200 píxeles cargada desde una URL de GitHub y, debajo de la imagen, un texto descriptivo con el nombre de la sección en color azul, asegurando que todos los imports entre carpetas sean técnicamente correctos para funcionar al instante.
